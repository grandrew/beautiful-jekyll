We offer:

  • Untrusted site

Junk Systems provides two interfaces:

  1. A Python 2.5+ script with no dependencies for x86-64 Linux that opens your system for SSH logins (Node Owner)
  2. An HTTP-GET call to get login credentials in JSON format
